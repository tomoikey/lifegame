# なぜ作ったのか？

可愛いから

# インストール方法

```sh
❯ git clone https://github.com/tomoikey/lifegame.git
❯ cd lifegame
❯ cargo build --release
❯ (あとはよしなに)
```

# どうやって使うのか？

```sh
❯ lifegame --help
Usage: lifegame [OPTIONS]

Options:
  -r, --ratio <RATIO>                        [default: 0.12]
  -m, --millis-per-frame <MILLIS_PER_FRAME>  [default: 100]
  -h, --help
```
